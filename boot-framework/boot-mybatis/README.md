# boot-mybatis  使用指引

    orm框架           mybatis-plus 3.1.1
    mysql驱动         mysql 8.0.16
    数据库连接池      druid 1.1.18
    sql打印           p6spy 3.8.1
    时间工具          joda-time 2.10.2
    1、MybatisPlus配置
    2、entity、dto、po、vo
    3、字典枚举封装
    4、雪花算法uuid生成
    5、p6spy sql执行打印
    6、重写spring的BeanUtils和StringUtils
